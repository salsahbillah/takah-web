# TAKAH

Frontend Smart Letter Management System built with React, Vite, Tailwind CSS, and Golang REST API.

---

# Tech Stack

- React 19
- Vite
- Tailwind CSS 4
- Axios
- React Router DOM
- Golang Gin REST API
- MySQL
- JWT Authentication
